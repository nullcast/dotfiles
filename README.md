# dotfiles

&emsp;

## Overview

This repository contains scripts to set macOS.

&emsp;

## Required

- [macOS](https://www.apple.com/jp/macos/monterey/) - v12.4

&emsp;

## Install

Set macOS.

```shell
$ brew tap-new my/casks
$ ln -s (brew --repo my/casks)/* .bin/.config/brew
$ cd ~/dotfiles && make
```
